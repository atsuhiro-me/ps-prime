# ps-prime
A PostScript file to generate prime numbers

## Usage

Print `pp.ps` on a PostScript compatible printer. If you don't have one, consider using [PDF24 Creator](https://tools.pdf24.org/) or [Adobe Acrobat Distiller](https://helpx.adobe.com/jp/acrobat/using/creating-pdfs-acrobat-distiller.html).

Change `/numpages 5 def` to any integer to set desired page number.

## Samples

`pp5.pdf` and `pp10.pdf` are samples pdf files rendered by PDF24.
